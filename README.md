Task Manager App

A task management app with user authentication.

## Setup

Backend
1. `cd backend`
2. `npm install`
3. Create `.env` file with MongoDB URL
4. `npm run dev`

Frontend
1. `cd frontend`
2. `npm install`
3. `npm start`

Technologies
- React.js
- Node.js
- Express.js
- MongoDB
- JWT

Features
- Register/Login
- Add/Edit/Delete tasks
- Mark tasks complete
- Search & filter tasks
